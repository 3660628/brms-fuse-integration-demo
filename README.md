JBoss BRMS & JBoss Fuse Integration Demo
========================================

Demo based on JBoss BRMS/Fuse products.


Quickstart
----------

1. Clone project.

2. Add products to installs directory.

3. Run 'init.sh'

4. Enjoy the demo!
